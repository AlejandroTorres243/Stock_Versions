Precarga y cacheado

Precarga

En los propios enlaces se puede utilizar data-fecth como atributo para precargar las paginas

La forma "programatica" de hacer esto es $.mobile.loadPage( pageUrl, { showLoadMsg: false } );

Mucho cuidado que esto aumenta el ancho de banda

  • Enlaces que seran cargados via Ajax con transiciones de pagina
  • Enlace al mismo dominio
  • Enlace a un dialogo: data-rel="dialog" (no almacenado en el historial del documento)

Cacheado

$.mobile.page.prototype.options.domCache = true;

<div data-role="page" id="cacheMe" data-dom-cache="true">

pageContainerElement.page({ domCache: true });

Ojo que si se hace esto mucho, el dom puede ser muy grande y dar problemas de rendimiento.
