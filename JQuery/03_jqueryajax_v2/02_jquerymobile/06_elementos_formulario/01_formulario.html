Inicializacion de elementos

Por defecto jquery mobile inicializa los elementos de forma automatica para darles ese aspecto

Si necesitas incrustar elementos de forma dinamica puedes lanzar el evento create para lanzar la auto-inicializacion Se puede lanzar en cualquier elemento (por ejemplo el div page) Para que no tengas que inicializar cada elemento

Por ejemplo esto seria para incrustar un formulario de login que nos pasen desde AJAX como codigo HTML

$(" El nuevo selector con los widgets/elementos de formulario") .appendTo( ".ui-page" ).trigger( "create" );

Refrescando elementos

Checkboxes:

$("input[type='checkbox']").prop("checked",true).checkboxradio("refresh");

Radios:

$("input[type='radio']").prop("checked",true).checkboxradio("refresh");

Selects:

var myselect = $("#selectfoo");
myselect[0].selectedIndex = 3;
myselect.selectmenu("refresh");

Sliders:

$("input[type='range']").val(60).slider("refresh");

Flip switches (they use slider):

var myswitch = $("#selectbar");
myswitch[0].selectedIndex = 1;
myswitch.slider("refresh");
